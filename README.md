# ondine.dev

Landing page for [Ondine](https://github.com/ptimizeroracle/ondine) -- The LLM Dataset Engine.

## Development

```bash
# Preview locally
npx serve src

# Or just open src/index.html in a browser
```

## Deployment

Auto-deploys to Cloudflare Pages / Vercel on push to `main`.
